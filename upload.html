💡 How this works
  1. 1Must be served from a PHP-enabled web server (Apache/Nginx). Opening via file:// will not work.
  2. 2On load, existing repository.txt entries appear automatically in the list on the right.
  3. 3Fill in the form, attach your file, and click Add to Repository.
  4. 4The file is uploaded to the correct folder and repository.txt is updated — all automatically via save.php.
  5. 5Use Download .txt any time as a manual backup.
